## Chip8 Interpreter
Just another [CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) interpreter.

---
## Controls
    Pause      - Spacebar
    Close      - Escape
    Debug      - F1
    Fullscreen - F2
    Borders    - F3

    1234   123C
    QWER   456D
    ASDF   789E
    ZXCV   A0BF

---
## Usage
Supply the path to the desired ROM as a command line argument.
>     ./main <path>

---
## Accuracy
Uncertain - but I think it passes all the test ROMs I could find.

---
## Requirements
SDL2
